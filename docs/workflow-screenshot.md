# Workflow Screenshot

Save your Copilot Studio workflow screenshot here as `workflow-screenshot.png`.

The screenshot should show the 3-node workflow:

```
[Start] → [Agent] → [Queue a new build]
```

## How to Capture

1. Open your workflow in Copilot Studio
2. Zoom to fit all nodes
3. Take a screenshot (Windows: Win+Shift+S)
4. Save as `workflow-screenshot.png` in this folder

## Example Layout

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│    ⚡ Start     │────▶│    🤖 Agent     │────▶│  📤 Queue a     │
│                 │     │                 │     │    new build    │
└─────────────────┘     └─────────────────┘     └─────────────────┘
```
